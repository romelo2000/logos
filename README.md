# LOGOS V27+ — AI Reality Engine

LOGOS — AI-система на базе принципов Transurfing. Работает локально через Ollama. Без облака, без подписок, без слежки.

## Скачать

[Скачать DMG для Mac (Apple Silicon)](https://github.com/romelo2000/logos/releases/latest)

## Требования

- macOS 12+ (Apple Silicon)
- 16 GB RAM
- 25 GB свободного места
- Интернет для первичной установки моделей

## Архитектура

- 32 числовые метрики (28 базовых + 4 кармических)
- 5 внутренних сил
- 16 рефлексов (12 базовых + 4 кармических)
- 6 якорей ответов (5 базовых + КАРМА)
- 3 режима: Fast, Deep, Critical
- 7 архетипов

## Кармический слой

- **karmicDebt** — накопленный кармический долг (0–1)
- **karmicMomentum** — направление текущего выбора (-1…+1)
- **karmicResonance** — выравнивание с целью души (0–1)
- **karmicFriction** — сопротивление старых паттернов (0–1)

Четыре кармических рефлекса: ВОЗВРАТ, РАСПЛАТА, СЕМЯ, ОЧИЩЕНИЕ.

## Модели

- **Fast**: Qwen 2.5 7B Instruct (~4.7 GB)
- **Deep**: Qwen 3 30B A3B Thinking (~18 GB)

Обе модели запускаются локально через Ollama.

## Лицензия

10 дней триал (полнофункциональный), затем активация по ключу с привязкой к Mac.
