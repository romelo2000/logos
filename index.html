Что умеет LOGOS

Не отвечает на вопросы — работает с состоянием

🧭

5 внутренних сил

Важность, Намерение, Активность, Покой, Контроль. Система отслеживает баланс сил в реальном времени и отвечает из текущего состояния, а не из шаблона.

📊

28 числовых метрик

Каждый ответ оценивается по 28 параметрам. Не текст — числа. Эмерджентное поведение возникает из числового состояния, а не из явных инструкций.

🔄

16 рефлексов

Автоматические реакции системы: резонанс, сдвиг, отражение, напряжение. Рефлексы срабатывают при определённых состояниях метрик — как рефлексы живого организма.

⚓

Якоря ответов

Каждый ответ содержит якоря: ОТРАЖЕНИЕ, ОПОРА, СДВИГ, НАПРЯЖЕНИЕ, ОТКРЫТИЕ. Не заголовки — точки опоры для восприятия.

🧠

Память сессий

LOGOS помнит ваши предыдущие сессии, обнаруживает паттерны и адаптирует ответы. Память хранится локально — только у вас.

⚡

Три режима

Fast (Qwen 2.5 7B) — быстрые ответы. Deep (Qwen 3 30B) — глубокий анализ. Critical — критическая оценка. Переключение на лету.
